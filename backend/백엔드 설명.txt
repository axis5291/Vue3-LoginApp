npm i body-parser 를 설치하여 프로트엔드에서 보낸 값을 받는 역할을 설치한다.
이로써, 백엔드에서 request.body를 사용할 수 있다.

//cors관련
프론트엔드와 백엔드 서버가 물리적으로 동일하거나 분리되어 있어도,
✔ 같은 도메인명이라도 프로토콜(http, https) 또는 포트번호(3000, 5000)가 다르면
✔ 브라우저의 보안 정책(CORS)이 실행되어 요청이 차단된다.
✔ 이를 해결하기 위해 백엔드 서버에서
const cors = require('cors'); 를 사용하여 CORS 정책을 허용하면 문제 해결 가능하다.

//새로고침 이슈
로그인한 후 새로고침을 하면 로그인이 풀리게 되는데 이를 해결위해선 백앤드 서버에서 쿠키 설정이 필요하다.
npm install cookie-parser 설치가 필요하다
